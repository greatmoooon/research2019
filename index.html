今年はお金をつかわず無料の物ばかりでホームページを作りました。
テキストエディタはVisual Studio Codeを使い、HTMLのアップロード先は GitHub pagesを使いました。


GitHub上に新しくリポジトリを作る。

  1. VScodoに先に作ったリポジトリを登録する。
  2. リポジトリ上にファイルを作ってcommitする。
  3. commitを繰り返し出来上がったファイルをGitHubへpushする。
　　

gitub上で新しくリポジトリを作る順番

  1. GitHubにログインする。
  2. 上のRepositoriesタブをクリックする。
  3. 移動したページで右上のNewをクリックする。
  4. 移動したページで次の項目を入力する。
    1. Repositorty name：resarch2019
    2. Description:2019年 夏の自由研究
    3. 公開範囲：Public
  5. create Repositoryをクリックしリポジトリを作る。
  6. 表httpcとsshのコマンドラインを控えておく。

VScodeにリポジトリを登録する順番

  1. ワークスペースにresarch2019と言うディレクトリを作り、そこへ移動 する。
  2. 左側にあるソース管理のアイコンをクリックする。
  3. 上部に出るアイコンによりディレクトリをリポジトリとして使えるように 初期化する。
  4. command+shift+pでコマンドラインを呼び出す。
  5. コマンドラインにgit add Remoteを入力して、先に控えたhttpsを入力 する。

commitする順番

  1. リポジトリとして作成したディレクトリにいる事を確認する。
  2. ファイルを新規作成しindix.htmlという名前で保存する。
  3. 左帯のソース管理のアイコンをクリックする。
  4. 表示されるファイル名の横にあるプラスマークをクリックしリポジトリにファイル を追加する。
  5. ファイルのstatusがUからAに変わる。
  6. メッセージ欄に初めてのコミットと入力し、上部のチェックマークをクリックして commitする。

pushする順番

  1. ソース管理の選択コマンドの中からpsuhを選択する。
  2. 先に登録したGitHudの公開先を選択すればユーザーとパスワードを聞かれるので入力 する。
  3. GitHub上でリポジトリがアップされていることを確認する。
  4. GitHubで->Settings->GitHub Pagesの項目へと進む。
  5. Sourceの項目でmaster branchを選択する。
  6. 指定される https://greatmoooon.github.io/research2019/をブラウザで表示すると ホームページが見られる。
